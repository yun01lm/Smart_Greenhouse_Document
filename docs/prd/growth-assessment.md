## 概述

作物长势评估模块利用大棚内全天候监控摄像头定时抓取植株图像，通过AI视觉分析技术识别作物所处生长阶段，评估株高、叶面积、叶色等长势特征。该模块支持长势历史对比，关联作物生长周期信息，提供生长时间线视图，完整追溯作物从种植到收获的生长全过程。

**覆盖端**：APP端（Android）

**使用角色**：
- 棚主(OWNER)：查看自己大棚作物的长势评估
- 员工(WORKER)：查看被授权大棚作物的长势评估

## 用户故事

### US-001 摄像头定时截帧查看

> 作为大棚棚主，我希望在APP中查看监控摄像头定时抓取的植株图像，无需亲自去大棚就能远程观察作物状态。

**验收条件**：
- 展示最近一次摄像头截帧图像
- 显示截帧时间戳
- 支持手动触发截帧（即时刷新）
- 显示截帧来源摄像头标识

### US-002 生长阶段识别

> 作为大棚棚主，我希望系统自动识别作物当前所处的生长阶段（如苗期、开花期、结果期等），以便根据不同阶段调整管理措施。

**验收条件**：
- 展示当前识别到的生长阶段名称
- 显示阶段识别置信度
- 提供该阶段的特征描述和管理要点
- 支持手动修正识别结果

### US-003 长势特征评估

> 作为大棚棚主，我希望系统能定量评估作物的长势特征（株高、叶面积、叶色等），以便客观判断作物生长状况。

**验收条件**：
- 展示株高估计值（cm）
- 展示叶面积指数估计
- 展示叶色健康度评分（0-100）
- 各指标附带与同期正常值的对比（偏高/正常/偏低）

### US-004 长势历史对比

> 作为大棚棚主，我希望查看作物不同时期的长势对比，直观了解作物生长变化。

**验收条件**：
- 支持选择两个时间点的长势评估进行对比
- 并排展示两张截帧图像
- 对比展示各长势指标的变化量
- 支持滑动时间轴选择对比时间点

### US-005 多模态融合健康综合评分

> 作为大棚棚主，我希望看到一个综合了环境数据和视觉长势数据的健康评分，快速判断作物整体健康状况。

**验收条件**：
- 展示0-100分的综合健康评分
- 评分综合考虑环境时序特征和视觉长势特征
- 展示各子维度评分明细（环境健康度、长势健康度等）
- 健康评分随时间变化的趋势曲线

### US-006 当前种植周期信息展示

> 作为大棚棚主，我希望在长势评估页面看到当前种植周期的基本信息，了解作物品种、种植日期和已生长天数。

**验收条件**：
- 展示作物名称和品种
- 展示种植日期和已生长天数
- 展示当前所处的生长阶段
- 信息来源于crop_cycles表

### US-007 生长时间线视图

> 作为大棚棚主，我希望按时间轴查看作物从种植到现在的完整生长历程，包括各阶段的长势评估记录和预警记录。

**验收条件**：
- 时间轴从种植日期开始到当前日期
- 时间轴上标注各生长阶段的分界点
- 点击时间轴节点可查看该时间点的长势评估详情
- 时间轴上同时标注预警事件（不同颜色标识）
- 支持缩放和滚动浏览

## 验收标准

### 功能验收

- [ ] 摄像头截帧图像清晰可辨，定时截帧间隔可配置（默认30分钟）
- [ ] 手动触发截帧后30秒内返回新图像
- [ ] 生长阶段识别支持至少4个阶段（苗期、生长期、开花期、结果期）
- [ ] 长势特征评估输出株高、叶面积、叶色三个指标
- [ ] 历史对比功能支持选择任意两个历史时间点
- [ ] 健康综合评分计算逻辑透明，各子维度可查看
- [ ] 种植周期信息正确关联crop_cycles表
- [ ] 生长时间线展示完整的生长记录和预警事件

### 性能验收

- [ ] 截帧图像加载时间 < 2秒
- [ ] 生长阶段识别响应时间 < 5秒
- [ ] 长势评估响应时间 < 8秒
- [ ] 健康评分计算响应时间 < 3秒

### AI准确率验收

- [ ] 生长阶段识别准确率 > 85%
- [ ] 叶色健康度评分与实际观察一致率 > 80%
- [ ] 健康综合评分与人工评估偏差 < 15分

## 备注

1. **摄像头配置**：使用ESP32-CAM模块，运行RTSP推流固件。服务器端通过FFmpeg定时截帧（可配置间隔，默认30分钟）。截帧图像存储到文件服务器，同时在数据库中记录截帧元数据。

2. **长势特征评估模型**：
   - 株高估计：基于参考标尺的视觉比例估算（需在摄像头视野内放置已知尺寸的标尺）
   - 叶面积指数：基于绿色像素占比和冠层覆盖度估算
   - 叶色健康度：基于HSV色彩空间中绿色通道分析

3. **生长阶段定义**（以番茄为例，不同作物需配置不同阶段参数）：
   - 苗期：0-30天
   - 生长期：30-60天
   - 开花期：60-80天
   - 结果期：80-120天

4. **多模态融合评分算法**：
   - 环境健康度（40%权重）：基于温湿度/光照/CO₂等参数的偏离度
   - 长势健康度（40%权重）：基于株高/叶面积/叶色的评估
   - 生长阶段匹配度（20%权重）：实际长势与预期长势的偏差

5. **生长时间线数据来源**：
   - 生长阶段变更记录（crop_cycles）
   - 长势评估记录（growth_assessments表）
   - 预警记录（alerts表，仅展示严重和警告级别）

6. **申报书对应**：本模块对应申报书五大研究内容之"视觉AI长势评估与病虫害诊断"的长势评估部分，以及创新点三"环境时序特征与视觉长势特征融合分析"。
