## 概述

设备控制模块为用户提供大棚内执行设备的远程控制能力，包括单个设备的手动开关控制和基于预设场景的一键联动操作。支持的设备类型包括通风风机、电动卷帘、遮阳网、滴灌阀门、补光灯。该模块支持按传感器组区域进行设备控制，实现精准的区域化管理。

**覆盖端**：APP端（Android）+ Web管理端（设备管理）

**使用角色**：
- 棚主(OWNER)：控制自己大棚的全部设备
- 员工(WORKER)：控制被授权大棚和被授权功能的设备
- 管理员(ADMIN)：查看全部设备状态（Web端设备管理）

## 用户故事

### US-001 单个设备手动控制

> 作为大棚棚主，我希望能够单独控制某个设备（如打开补光灯），以便根据实际需要进行精确操作。

**验收条件**：
- 设备列表展示设备名称、类型、当前状态（开/关/运行中）
- 每个设备有独立的开关按钮
- 点击开关后设备状态实时更新
- 操作前无需确认（快速操作），但操作后显示结果反馈

### US-002 一键场景联动

> 作为大棚棚主，当遇到特定场景时（如高温天气），我希望一键触发多个设备的联动操作，而不是逐个操作设备。

**验收条件**：
- 提供预设场景按钮（如"高温通风"、"阴天补光"、"缺水灌溉"）
- 每个场景按钮显示将执行的设备操作列表
- 点击场景按钮后，相关设备按预设逻辑自动执行
- 场景执行过程中显示进度状态

### US-003 设备实时状态反馈

> 作为大棚棚主，发出控制指令后，我希望确认设备是否真的执行了操作。

**验收条件**：
- 控制指令下发后，设备状态在3秒内更新
- 如果设备未响应（离线/故障），显示错误提示
- 设备状态图标清晰区分：在线-运行中（绿色）、在线-停止（灰色）、离线（红色）
- 显示设备最后心跳时间

### US-004 控制执行结果确认

> 作为大棚棚主，我希望确认控制操作是否成功执行。

**验收条件**：
- 控制指令下发后显示Toast提示（"正在执行..."→"执行成功"/"执行失败"）
- 执行失败时显示失败原因（设备离线/指令超时）
- 支持重试操作

### US-005 支持的设备类型

> 作为大棚管理人员，系统应支持大棚内常见的执行设备控制。

**验收条件**：
- 支持控制通风风机（开关控制）
- 支持控制电动卷帘（开关/开度控制）
- 支持控制遮阳网（开关控制）
- 支持控制滴灌阀门（开关控制）
- 支持控制补光灯（开关/亮度控制）

### US-006 按传感器组区域控制设备

> 作为大棚棚主，大棚不同区域安装了不同的设备，我希望选择特定区域后控制该区域的设备。

**验收条件**：
- 设备列表支持按传感器组区域筛选
- 选择区域后仅显示该区域的设备
- 设备卡片上显示所属区域标注
- 场景联动支持按区域执行

## 验收标准

### 功能验收

- [ ] 设备列表正确展示所有已绑定设备及状态
- [ ] 单个设备开关控制响应时间 < 3秒
- [ ] 设备状态反馈准确（与实际设备状态一致）
- [ ] 场景联动触发后所有关联设备按预设逻辑执行
- [ ] 控制失败时显示明确的错误信息
- [ ] 5种设备类型全部支持
- [ ] 按区域筛选设备功能正常

### 性能验收

- [ ] 控制指令从APP到ESP32延迟 < 2秒（局域网）
- [ ] 设备状态回传延迟 < 3秒
- [ ] 场景联动中多设备控制时间差 < 1秒

### 安全验收

- [ ] 设备控制需校验用户权限（棚主可控制全部，员工仅被授权设备）
- [ ] 控制指令需经过后端校验，不可前端绕过
- [ ] 操作日志完整记录（谁、何时、控制了什么设备、结果）

### 可靠性验收

- [ ] 设备离线时有明确的状态标识
- [ ] 控制指令下发后设备无响应时有超时处理（30秒超时）
- [ ] 支持控制指令重试（最多3次）

## 备注

1. **MQTT Topic设计**：
   - 控制指令下发：`greenhouse/{greenhouse_id}/group/{group_id}/device/{device_id}/control`
   - 设备状态回传：`greenhouse/{greenhouse_id}/group/{group_id}/device/{device_id}/status`
   - 控制指令格式：`{"action": "on"/"off"/"set", "value": 80, "timestamp": "..."}`

2. **设备类型与控制方式**：
   | 设备类型 | 控制方式 | 控制引脚 |
   |----------|----------|----------|
   | 通风风机 | 开关 | GPIO继电器 |
   | 电动卷帘 | 开关+开度 | GPIO继电器（正反转+限位开关） |
   | 遮阳网 | 开关 | GPIO继电器 |
   | 滴灌阀门 | 开关 | GPIO继电器（电磁阀） |
   | 补光灯 | 开关+亮度 | GPIO继电器 + PWM |

3. **场景联动预设**（初期）：
   | 场景 | 触发条件 | 执行动作 |
   |------|----------|----------|
   | 高温通风 | 温度 > 35°C | 开启风机 + 开启卷帘（50%） |
   | 阴天补光 | 光照 < 5000 lux | 开启补光灯（80%亮度） |
   | 缺水灌溉 | 土壤湿度 < 30% | 开启滴灌阀门（15分钟） |
   | 夜间保温 | 时间 18:00 | 关闭卷帘 + 关闭遮阳网 |

4. **设备安全保护**：
   - 电机类设备（卷帘）需有运行时间保护（超过最大运行时间自动停止）
   - 滴灌阀门需有最长开启时间限制（防止漏水）
   - 补光灯需有最小关闭间隔（防止频闪损坏）

5. **操作日志格式**：
   ```
   {
     user_id, device_id, action, value,
     scene_id(可选), result, error_message, created_at
   }
   ```

6. **申报书对应**：本模块对应申报书五大研究内容之"设备一键联动控制"。
